미디어 관련 태그

html에는 미디어를 표현하기 위한 이미지, 비디오, 오디오 태그가 존재합니다.
따라서, 미디어를 표현하기 위한 <image>,<audio>,<vidio>태그에 대해 알아봅시다.

1. <image> 태그

웹 페이지에 이미지를 넣을 때에는 <image> 태그를 사용합니다.
이미지 태그 하나당 1개의 이미지를 삽입할 수 있으며, 반드시 src 속성을 사용해 이미지의 경로를 지정해야 합니다.

        
            <img src="img_url">

해당 태그 사용의 결과는 다음과 같습니다.

이미지 태그의 속성

이미지 태그는 다양한 속성이 있습니다. 속성은 아래의 표를 참고해 주세요.

attribute Description
width 너비
height 높이
alt 이미지 대체텍스트
title 이미지의 제목

2. <audio> 태그

<audio> 태그는 음악을 삽입하는 태그입니다. 사용 방식은 img 태그와 동일합니다.

오디오 태그의 속성

오디오 태그는 다양한 속성이 있습니다. 속성은 아래의 표를 참고해 주세요.

attribute Description
src 음악 파일 경로
preload 재생 전 음악 파일을 모두 불러올 것인지 지정
loop 오디오 반복 재생 설정

3. <vidio> 태그

<vidio> 태그는 비디오를 삽입하는 태그입니다. 사용 방식은 img 태그와 동일합니다.

비디오 태그의 속성

비디오 태그는 다양한 속성이 있습니다. 속성은 아래의 표를 참고해 주세요.

attribute Description
src 동영상 파일 경로
preload 재생 전 동영상 파일을 모두 불러올 것인지 지정
poster 동영상 준비 중 표시될 이미지 파일 경로
loop 동영상 파일 반복 지정
width 너비
height 높이
