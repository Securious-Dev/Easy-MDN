1. 리스트

html에는 다양한 목록 관련 태그가 존재합니다. 해당 페이지에서는 목록을 표현하기 위한 태그에 대해 알아보도록 합시다.
대표적으로 목록을 나타내기 위한 태그인 <ul>, <ol>, <li>에 대해 알아보고자 합니다.

1.1. <ul> & <li> 태그

<ol> 태그는 순서있는 목록을 만들기 위해 주로 사용합니다.
내부에 list의 요소처럼 순서가 없는 목록을 만들 때 사용합니다.
주로, ul 태그와 li태그는 다음과 같이 함께 사용하며, 원과 같은 불릿으로 표현이 됩니다.

        
            <ul>
                <li>HTML</li>
                <li>CSS</li>
                <li>JS</li>
            </ul>

해당 태그 사용의 결과는 다음과 같습니다.

        
            
            
    
                
  • HTML
    • 
                
  • CSS
    • 
                
  • JS
    •

1.2. <ol> 태그

<ol> 태그는 이전과는 다르게 순서를 나타낼 때 사용하는 태그입니다.

        
            <ol>
                <li>HTML</li>
                <li>CSS</li>
                <li>JS</li>
            </ol>
