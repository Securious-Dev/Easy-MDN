1. 종류와 중요도

html에는 사용하고자 하는 종류와 중요도에 따라 태그를 달리하여 나타낼 수 있습니다.
대표적으로 강조하기 위한 태그인 <b>, <strong>, <i>, <em>에 대해 알아보도록 합시다.

1.1. <b> 태그

<b> 태그는 단순히 글자를 굵게 만드는 용도입니다.

      
        <b> b tage usage </b>;

해당 태그 사용의 결과는 다음과 같습니다.

        
          b tag usage

1.2. <strong> 태그

<strong> 태그는 b 태그와 달리 단순히 글자를 굵게 만드는 것에서 더 나아가 해당 내용이 중요하다는 의미를 담고 있습니다.

        
            <strong> strong tage usage </strong>;

해당 태그 사용의 결과는 다음과 같습니다.

        
            strong tag usage

1.3. <i> 태그

<b> 태그는 단순히 글자를 기울이게 만드는 용도입니다.

        
          <i> i tage usage </i>;

해당 태그 사용의 결과는 다음과 같습니다.

      
        i tag usage

1.4. <em> 태그

<em> 태그는 단순히 글자를 기울이는 것에서 더 나아가 strong 태그와 마찬가지로 강조의 역할을 합니다.

      
        <em> em tage usage </em>;

해당 태그 사용의 결과는 다음과 같습니다.

      
        em tag usage
