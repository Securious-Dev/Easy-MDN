2. JavaScript 심화 문법

2.1. 반복문

반복문은 여러 동작을 반복하는 경우 사용합니다. 반복문을 사용하게 되면 동일한 코드를 여러 번 반복할 수 있습니다.

반복문을 사용하는 방법에는 for문과 while문이 있습니다.

2.1.1. while문

    
      while (조건) {
        // 조건을 만족하는 경우 실행할 코드
      }

while문은 조건을 만족하는 경우 계속 실행됩니다.

조건이 참인 경우 계속 실행되기 때문에 무한 루프를 만들 수 있습니다.

    
      while (true) {
        // 무한히 실행할 코드
      }

위의 코드에서 while문의 조건이 항상 true 이므로 해당 while문은 무한히 실행됩니다.

대부분의 while문은 조건을 만족 시키는 경우를 while문 안에 삽입하여 사용합니다.

    
      let i = 1;
      while (i < 11) {
        console.log(i);
        i++;
      }

i의 경우 코드가 실행되며 계속 값이 변화하기 때문에 let 키워드로 선언해야 합니다.

위 코드의 결과는 1부터 10까지의 숫자가 콘솔로 출력됩니다. (콘솔창을 열어 결과를 확인해보세요)

i가 11이 되면 while문의 조건을 만족하지 않기 때문에 코드가 실행되지 않습니다.

2.1.2. for문

while문 보다 복잡한 형태를 갖지만 가장 많이 쓰이는 반복문 입니다.

    
      for (반복문 시작점; 반복문 조건; 각 반복 이후 수행될 것) {
        // 조건을 만족하는 경우 실행될 코드
      }

마찬가지로 1부터 10까지 콘솔창에 출력하도록 하는 for문은 다음과 같이 작성할 수 있습니다. (콘솔창을 열어 결과를 확인해보세요)

    
      for ( let i = 1; i < 11; i++ ) {
        console.log(i);
      }

i는 1부터 시작합니다. i가 1인경우 i < 11이라는 조건을 만족하기 때문에 for문 안에 있는 코드가 실행됩니다.

for문 안의 코드가 실행되고 난 뒤, i++을 수행하여 i는 2가 됩니다.

위의 과정을 반복하여 i가 11이 된다면 i < 11을 만족하지 않기 때문에 더이상 for문은 실행되지 않습니다.

결론적으로 반복문 시작점은 for문이 처음 실행되며 한 번만 실행되며 for문이 반복되는 동안 반복문의 조건과 각 반복 이후 수행될 것은 계속 실행됩니다.

for문에서 무한루프는 아래와 같이 만들 수 있습니다.

    
      for (;;) {
        // 무한히 실행할 코드
      }
