1. JavaScript 시작하기

1.1. JavaScript란?

자바스크립트는 여러분의 웹사이트에 상호작용성(예를 들면, 게임, 버튼이 눌리거나 폼에 자료가 입력될 때 반응, 동적인 스타일링과 애니메이션)을 더해 주는 프로그래밍 언어 입니다.


2. JavaScript 기본 문법

2.1. 변수 선언

JS에서 변수를 만들기 위해서는 const, let, var 키워드를 사용합니다.

    
      const name = 'Kim';
      let age = 25;

const는 중간에 변화하지 않는 변수, let은 중간에 변화할 수 있는 변수를 선언하는데 사용하며, var는 거의 사용하지 않습니다.

따라서, 대부분의 변수는 const를 이용하여 선언하고 몇몇 경우에만 let을 사용하는 것을 권장합니다.

2.2. 주석

주석은 쉽게 생각하면 코드에 남기는 메모입니다.

자신이 작성한 코드에 관한 정보 등을 기록할 수 있으며 본인 또는 타인이 코드를 이해하는데 도움을 줄 수 있습니다.

주석은 코드의 실행에 영향을 미치지 않기 때문에 자유롭게 작성할 수 있습니다.

    
      const name = 'Kim; // 사람의 이름을 담는 변수
      // let age = 25;
      
      /* 여러줄의 주석을 작성하는 경우에는
      이와 같이 작성할 수 있습니다. */

위의 예시에서 두번째 줄의 코드는 주석이기 때문에 실행되지 않습니다.

위와 같이 주석은 //를 이용하여 작성할 수 있습니다.

만약 여러 줄의 주석을 작성하고자 한다면 /*와 */안에 여러줄로 구성된 주석을 작성할 수 있습니다.

2.3. 세미콜론

JS에서 세미콜론이 자주 사용되는 것을 볼 수 있습니다.

JS에서 세미콜론은 문장의 종료를 나타냅니다. JS가 작동하는 동안 컴퓨터는 세미콜론에 의해 문장의 끝을 파악합니다.

하지만, 세미콜론을 붙이지 않아도 코드가 정상 작동하는 것을 확인할 수 있습니다.

그 이유는 JS는 자동으로 문장의 끝이라고 "추측" 되는 곳에 세미콜론을 붙이기 때문입니다.

자동으로 붙여주지만 JS의 "추측"과 우리의 "의도"가 다를 수 있습니다. 따라서, 가급적 세미콜론을 붙이는 것을 추천드립니다.

(단, if (...) {...}, for (...) {...} 과 같이 0개 이상의 문장을 묶은 코드 블록 뒤에는 붙이지 않습니다.)

2.4. 출력

JS에서 결과를 출력하는 방법중 alert와 console.log에 대해 알아보겠습니다.

alert는 브라우저의 경고창을 이용해 출력합니다.

    
      window.alert('Hello World!'); // window.alert() 대신 alert()만 사용해도 작동한다.

console.log는 콘솔창을 이용해 출력합니다.

(콘솔창은 브라우저에서 F12 > 콘솔에서 사용할 수 있습니다. 이 콘솔창에 JS 코드를 작성하여 실행할 수 있습니다.)

    
      console.log('Hello World!');
    
     (버튼을 누르고 콘솔창을 확인하세요.)
